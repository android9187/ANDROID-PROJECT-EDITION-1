# Конвертер валют

Простой консольный конвертер валют с поддержкой 15+ валют.

## Функциональность
- Конвертация между 15 валютами
- Актуальные курсы с API
- Резервные курсы при отсутствии интернета
- Проверка правильности расчетов

## Требования
- Python 3.6 или выше

## Запуск
```bash
pip install -r requirements.txt
python currency_converter.py
